# ANALISISMULTIVARIAT_CBRS
Cluster Recommender Based System

Cluster Based Recommender System merupakan Sebuah sistem yang dirancang untuk memprediksi apa yang disukai oleh pengguna berdasarkan interaksi sebelumnya dan tindakan serupa.
contoh dalam kehidupan berupa Netflix merekomendasikan film berdasarkan riwayat tontonan, tokopedia merekomendasikan produk sesuai minat belanja.

data yang digunakan merupakan data Foodpanda yang diperoleh dari kaggle. hal ini terdapat 20 variabel. namun, pada CBRS yang digunakan hanya 8 variabel. 
berdasarkan hasil evaluasi dari silhouette score dan davies bouldin pada kedua clustering, kedua klaster termasuk ke golongan cukup baik. karena data ini tidak memiliki ground truth atau label.
